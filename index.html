Step Mempush code ke repo baru
  • 1. Siapkan file atau foleder yg mau di upload github
  • 2. Git init (inisiasi git, agar bisa diketahui oleh sistem bahwa, dalam folder tsb bisa di lakukan push (upload code))
  • 3. Upload file & folder

    - git remote add origin https://github.com/fshlarf/digidev-test-repo.git
    (hanya utk repo baru yg masih kosong)

    - git add .

    git commit -m "saya ngerjain index.html"

    git push (git push --set-upstream origin master)
